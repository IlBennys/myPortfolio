# Benny's Portfolio

## `npm start`

Dalla root directory del progetto "myPortfolio" (cd myPortfolio), dopo aver installato i node modules con il comando:
npm install o npm i

Entrare prima nella directory server e avviare il server, accertandosi tramite il log che sia in ascolto sulla porta 5000:
cd server
npm start

In un'altra console, dopo essere tornati nella root directory del progetto, digitare:
cd client
npm start

Digitare http://localhost:3000 nel browser per aprire il progetto.
